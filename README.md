# 🎌 JapLearn

An AI-powered Japanese learning web app built with React and Groq API.

## Features
- 🔤 Translate English to Japanese instantly
- 📖 See Kanji, Hiragana, Romaji and grammar explanation
- 💬 Get example sentences
- ⭐ Save words to your personal Review Deck
- 🃏 Flashcard-style review system

## Tech Stack
- React + Vite
- Groq API (LLaMA 3)
- CSS with inline styles
